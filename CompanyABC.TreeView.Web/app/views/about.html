Overview

This ASP.NET MVC web application renders a tree structure to a depth specified by the input.

Start playing

When the application is run (with the default configuration), you can access a tree of depth 2 by typing into the internet browser address bar the url .

Troubleshooting

Please ensure the "Start Action" for the web project (CompanyABC.TreeView.Web) has been set to "Specific Page" with a blank value.
