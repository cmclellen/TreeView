Overview

This ASP.NET MVC web application renders a tree structure to a depth specified by the input.

I've used Visual Studio 2012 along with ASP.NET MVC5 and AngularJs 1.4 (for it's ease of keeping concerns separate by providing an MVVM type pattern) to develop this application.

Start playing

When the application is run (with the default configuration), you can access a tree of depth 2 by either typing into the internet browser address bar or clicking on the url .

Troubleshooting

Please ensure the "Start Action" for the web project (CompanyABC.TreeView.Web) has been set to "Specific Page" with a blank value.

To ensure the JS tests run, you'll have to install Chutzpah for Visual Studio.
